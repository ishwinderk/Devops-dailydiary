Task - Setup jenkins
Open Powershell, run as administrator
1. docker run -p 8080:8080 -p 50000:50000 -d -v jenkins_home:/var/jenkins_home jenkins/jenkins:lts
2. docker ps (displays container_id)
3. docker logs container_id(password token will be displayed)
4. Now open in browser localhost:8080
5. enter password token and then log in.
