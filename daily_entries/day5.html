Local Machine -> Intermediate (Github) ->Target (Test/Production)
Once Docker Desktop is installed, create image on docker desktop. Follow the following steps:
1. Create a folder
2. Open in vscode editor
3. Create a file named app.js
4. Write inside the file - Console.log("Hi, Welcome to Docker!")
5. Open terminal and type command node app.js(Install node if previously not installed)

Create new file in the same folder named Dockerfile(write the following in this file)
1. FROM node:alpine
2. COPY ./app(name of file)
3. WORKDIR /app
4. CMD node app.js
Tip- Dockerfile vary language to language

save this file and open terminal
1. docker --version
2. docker build -t hello-docker .
3. docker image ls
4. docker run hello-docker

Task - 1. Push this image from local system into docker hub(hub.docker.com)
Steps are : i. docker image tag image_name:latest docker_username/image_name:latest
ii. docker login
iii. docker image push docker_username/image_name:latest

Other possible steps are:
i. docker login
ii. docker tag image_name:docker_username/repository_name:latest
iii. docker push docker_username/repository_name:latest

2. Test using Play-with-docker
i. Open play-with-docker
ii. Add new instance, editor will open
iii. command - docker run docker_username/image_name:latest (To pull, replace pull with run)
output- message written in the file app.js will be displayed!
